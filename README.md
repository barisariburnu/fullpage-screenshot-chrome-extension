## Keyboard Shortcuts

- Full-page screenshot: `Ctrl+Shift+F` (Mac: `Command+Shift+F`)
- Visible area screenshot: `Ctrl+Shift+V` (Mac: `Command+Shift+V`)

## Required Permissions

- `activeTab`: Essential for capturing screenshots of the active browser tab
- `downloads`: Required for local screenshot storage
- `debugger`: Necessary for full-page screenshot functionality
- `<all_urls>`: Required for cross-site functionality

## Security & Privacy

- Zero personal data collection
- Local-only storage implementation
- No internet connectivity required
- No third-party service dependencies

## Installation

1. Install "Full Page Screenshot" from the Chrome Web Store
2. Pin the extension from Chrome's extension menu
3. Click the extension icon on any webpage
4. Select "Full Page" or "Visible Area" capture option

## Development Setup

1. Clone the repository
```bash
git clone https://github.com/barisariburnu/fullpage-screenshot-chrome-extension.git
```

2. Enable Chrome Developer Mode
- Navigate to `chrome://extensions`
- Enable "Developer mode"

3. Load the Extension
- Click "Load unpacked"
- Select the project directory
